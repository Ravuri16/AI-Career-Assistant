# AI Career Assistant

## Overview
AI Career Assistant is a web-based application that helps job seekers evaluate their resumes against job descriptions.

The tool identifies matching skills, highlights missing keywords, and generates interview preparation guidance to help candidates improve their job applications.

## Features
- Resume analysis
- Job description matching
- Skill gap identification
- Interview question suggestions
- Career preparation assistance

## Technologies Used
- Python
- Streamlit

## How It Works
1. Paste your resume.
2. Paste a job description.
3. Click Analyze.
4. Review matched skills, missing skills, and interview preparation suggestions.
